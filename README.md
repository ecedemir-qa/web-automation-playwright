## Overview
This project is an end-to-end test automation framework built with Playwright for the Sauce Demo web application.  
It covers core user flows such as authentication, product selection, checkout process, and form validations using a clean Page Object Model structure.

---

## Tech Stack
- Playwright Test Runner  
- JavaScript (Node.js)  
- Page Object Model (POM) design pattern  
- Git & GitHub for version control  
- GitHub Actions CI for automated test execution  

---

## Test Scope
The following main scenarios are automated:

- User login (positive scenarios)  
- Adding product to cart  
- Checkout happy path (complete purchase flow)  
- Checkout form validation (empty required fields)
